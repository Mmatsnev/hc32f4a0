平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示MCU的SMC SRAM功能。

说明：
以字节/半字/字的方式，写入SRAM，然后读出数据，验证数据一致性

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
串口助手软件

================================================================================
使用步骤
================================================================================
1）通过USB线，连接J25 DAP端口
2）打开串口助手软件，选择DAP虚拟端口，端口配置如下参数
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
3）打开工程并重新编译；
4）下载运行程序，
5）查看LED灯状态：蓝色LED灯闪烁，红色LED灯熄灭
6）查看串口助手打印信息

================================================================================
注意
================================================================================
LED状态说明：
1）蓝色LED灯闪烁表示正在进行读写测试
2）红色LED灯亮表示读写数据不一致
