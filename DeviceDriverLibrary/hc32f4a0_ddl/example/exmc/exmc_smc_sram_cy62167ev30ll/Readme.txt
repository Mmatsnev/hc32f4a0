平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示MCU的SMC SRAM功能。

说明：
以字节/半字/字的方式，写入SRAM，然后读出数据，验证数据一致性

================================================================================
测试环境
================================================================================
测试用板:
---------------------
MS_HC32F4A0_LQFP176_050_MEM

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）SRAM cy62167ev30ll存储扩展板插入MS_HC32F4A0_LQFP176_050_MEM板J7
2）打开工程并重新编译；
3）下载运行程序，
4）查看变量m_u32ByteTestErrorCnt、m_u32HalfwordTestErrorCnt、
   m_u32WordTestErrorCnt，变量值为0

================================================================================
注意
================================================================================
变量值说明：
1）变量值为0，表示读写数据一致
2）变量值大于零，表示字节/半字/字读写时出错个数
