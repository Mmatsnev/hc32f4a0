功能描述
================================================================================
本样例主要展示EMB模块的端口输入刹车功能。

说明：
EMB端口为P16，EMB检测端口输入电平，
1、满足端口输入电平时，PWM输出指定电平，产生EMB中断，执行EMB中断服务回调函数
2、EMB中断回调函数内，等待按键释放，然后清除EMB状态，PWM正常输出

================================================================================
使用步骤
================================================================================
1）测试板J5-PB8、J5-PB9引脚与示波器连接，J1-PB2与J1-3V3相连
2）编译、下载并运行
3）观察示波器，示波器J5-PB8、J5-PB9通道显示周期脉冲（2Hz）
4）J1-PB2与J1-GND相连
5）观察示波器，J5-PB8、J5-PB9通道电平为低
6）J1-PB2与J1-3V3相连
7）观察示波器，示波器J5-PB8、J5-PB9通道显示周期脉冲（2Hz）
8）重复步骤3） ~步骤7）
