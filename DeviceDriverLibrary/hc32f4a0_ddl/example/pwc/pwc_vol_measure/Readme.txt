平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示基准电压、VBAT电压测量、VBAT电压检测等功能。

说明：
本样例设置使用ADC单元1来测量基站及VBAT电压，启动后连续测量基准及VBAT电压，并通过
串口打印电压信息；其中基准电压约为1.15V，VABT电压则为外部纽扣电池电压，之后一直
监测VBAT电压是否大于2V，是则LED_B亮起；反之，则LED_R亮起。

串口调试助手软件

================================================================================
使用步骤
================================================================================
1）使用USB线连接电脑及目标板；
2）在串口调试助手SecureCRT上打开USB对应串口，配置端口如下参数：
     波特率：115200
     数据位：8
     校验位：None
     停止位：1
3）打开工程并重新编译，启动IDE的下载程序功能，关闭IDE下载界面；
4）观察串口打印的电压信息，并根据Vbat实际电压结合观察LED亮灭情况。
