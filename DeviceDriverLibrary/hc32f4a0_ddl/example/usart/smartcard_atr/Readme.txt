平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示USART外设配置为智能卡模式时读取ATR数据。
说明：
无

无

---------------------
无

================================================================================
使用步骤
================================================================================
1）将智能卡插入测试板SOCKET。
2）打开工程并重新编译。
3）下载代码测试板，并且全速运行。
4）查看LED状态，蓝色LED灯亮。
5）查看缓存变量m_stcRxBuf，第一个数据十六进制值为0x3B或0x3F，与智能卡TS字符匹配。

================================================================================
注意
================================================================================
LED状态说明：
1）蓝色LED灯亮表示智能卡ATR第一个字节数据为0x3B或0x3F，符合预期
2）红色LED灯闪烁表示未插入智能卡
3）红色LED灯常亮表示USART初始化失败或者智能卡ATR第一个字节数据不等于0x3B，不符合
   预期
