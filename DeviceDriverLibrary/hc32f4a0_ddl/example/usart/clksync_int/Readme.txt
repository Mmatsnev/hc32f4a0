平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示USART外设配置为时钟同步模式时通过中断方式收发数据。

================================================================================
使用步骤
================================================================================
1）需两个测试板，按照如下方式连接：
   主机:UART6   从机:UART6
    J9_GND------J9_GND
    J9_RX ------J9_TX
    J9_TX ------J9_RX
    J9_CK ------J9_CK
2）打开工程clksync_int, 配置宏定义如下，重新编译，下载代码主机：
   #define CLKSYNC_DEVICE_MODE             CLKSYNC_MASTER_MODE
3）修改如下宏定义，并重新编译，下载代码从机。
   #define CLKSYNC_DEVICE_MODE             CLKSYNC_SLAVE_MODE
4）分别重新上电
5）按从机按键SW10
6）按主机按键SW10
7）主机和从机蓝色LED点亮，表示两个板子收发数据一致，符合预期

================================================================================
注意
================================================================================
LED状态说明：
1）蓝色LED灯亮，表示数据收发一致
2）红色LED灯亮，表示数据收发不一致
