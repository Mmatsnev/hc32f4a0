平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例为使用 I2C 外设读写 AT24C02 E2PROM 和禁止 I2C Fast ACK 功能的样例。
说明：
无

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）确认开发板上 E2PROM 硬件连接是否正确；
3）启动IDE的下载和调试功能，全速运行；
4）观察开发板上LED（D26）被点亮，E2PROM 读写成功。

================================================================================
注意
================================================================================
开发板上LED（D26）没有被点亮，则 E2PROM 读写失败。
