平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示ICG模块中BOR复位功能。

说明：
关于ICG相关配置需参照mian()函数中的描述去修改hc32f4a0_icg.h中的宏定义。

本样例设置硬件启动BOR且电压阈值为2.3V，正常启动后LED_R闪烁，调整供电电压低于阈值
进入复位状态，LED_R熄灭；再次调整供电电压高于阈值进入正常工作状态，LED_R常亮。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）按下复位按键复位芯片，LED_R闪烁；
4）将目标板供电电压调至2.2V，此时芯片进入复位状态，LED_R熄灭；
5）将电压调至3.3V，芯片退出复位并进入正常工作状态，检测是欠压复位，LED_R常亮。
