平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程展示了TIMERA的相位差计数功能，以单元9为例，实现了对正交编码器进行转速测量。

说明：
1. 宏“APP_TMRA_CLK”在本例程中是正交信号的时序，请根据实际情况定义。
2. A相信号输入引脚为PG4，B相输入信号输入引脚为PG5。
3. 例程以TIMER2作1秒定时（定时基本单位为100ms），用以获取1秒内TIMERA的计数时钟个数
  （正交编码器输出的编码信号），再结合设定的TIMERA的相位差计数倍数和正交编码器每转一周输
   出的正交编码信号个数，计算可得到正交编码器当前的转速（转每分钟）。
4. TIMERA为16位定时器，需注意在所设定的用以计算转速的单位时间内，不要发生计数溢出。

辅助工具:
---------------------

辅助软件:
---------------------
串口调试助手软件（115200bps，8N1）

================================================================================
使用步骤
================================================================================
1）将正交编码器的A相信号和B相信号分别连接至PG4和PG5；
2）如果需要用通过上位机观察程序运行情况，请将USB转串口工具正确地与开发板上的J28端口（靠近
   Micro USB接口）连接；如果不需要，请忽略此步骤；
3）打开工程并重新编译；
4）启动IDE的下载和调试功能；
5）运行程序，可断点观察或在串口调试助手软件中观察程序运行的情况。
