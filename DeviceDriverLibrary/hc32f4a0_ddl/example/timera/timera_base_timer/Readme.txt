平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程用TIMERA单元1的通道1实现1ms定时，并将定时时序输出至PA2。

说明：
1. TIMERA单元1到单元4的时钟源是PCKL0，TIMERA单元5到单元12的时钟源是PCKL1。例程中，PCLK0
   为200MHz，PCLK1为100MHz。在选择定时器单元时，注意根据对应的时钟源设置正确的比较值；
2. 将“APP_TMRA_USE_INTERRUPT”定义为非零，使用TIMERA的比较匹配中断；
3. 将“APP_TMRA_USE_HW_TRIG”定义为非零，使能硬件触发条件，触发TIMERA动作（启动、停止等）。
   例程中用PC0的上升沿启动TIMERA，用PC0的下降沿停止TIMERA。

================================================================================
使用步骤
================================================================================
1）将示波器探头连接至PA2；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）运行程序：如果使能了硬件触发条件，在PC0上制造一个上升沿，以启动TIMERA；
5）可看到示波器显示一路频率为500Hz的方波。
