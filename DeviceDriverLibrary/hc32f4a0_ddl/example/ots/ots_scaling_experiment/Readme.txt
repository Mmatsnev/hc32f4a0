平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程主要介绍了如何给OTS做定标实验，以获得参数K（温度斜率）和M（温度偏移）。

说明：
1. 例程同时在HRC 16MHz和XTAL 8MHz时钟源下做定标实验，计算K和M时，注意时钟对应；
2. 在每次触发实验后，例程默认会连续测量10次，并计算每次实验数据的中间变量A；
3. 例程会输出每次实验时读取到的寄存器的数据及中间变量A，最后会输出A的平均值；
4. 计算K和M时用A的平均值计算。

================================================================================
使用步骤
================================================================================
1）请将USB转串口工具正确地与开发板上的J28端口（靠近Micro USB接口）连接；
2）打开工程并重新编译；
3）下载程序到目标板；
4）将目标板放入高低温箱，设置高低温箱目标温度为25℃（T1），运行；
5）当高低温箱温度稳定在25℃左右10分钟后，按下并释放按键SW10，触发当前温度的定标实验；
6）保存当前串口调试助手输出的实验数据，并记录当前高低温箱的实际温度；
7）设置高低温箱目标温度为105℃（T2），运行；
8）当高低温箱温度稳定在105℃左右10分钟后，按下并释放按键SW10，触发当前温度的定标实验；
9）保存当前串口调试助手输出的实验数据，并记录当前高低温箱的实际温度；
10）根据手册描述的公式，计算参数K（温度斜率）和M（温度偏移）。
