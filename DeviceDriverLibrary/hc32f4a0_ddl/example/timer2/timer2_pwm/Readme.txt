平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程用Timer2单元1的通道A输出一路频率为1MHz的PWM。PWM输出引脚为PA2。

说明：
1. 系统时钟为XTAL（8MHz）；
2. 不同的通道，对应不同的PWM输出引脚，并且PWM引脚的功能编号可能不同；
3. PWM频率最高为Timer2时钟源的一半；
4. PWM占空比恒为50%；
5. 同一个通道，如果有多个PWM输出引脚，可同时输出多路相同的PWM。

================================================================================
使用步骤
================================================================================
1）将示波器探头连接至PA2；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）运行程序，可看到示波器显示一路频率为1MHz的方波。
