平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程主要功能如下，通过“APP_TMR2_FUNC”选择：
1. 对其他外设事件进行计数（本例程中的计数对象为按键SW10产生的事件）；
2. 对TIMER6上溢事件或下溢事件计数（本例程中的计数对象为TIMER6单元1的上溢事件，上溢周期
   为10毫秒）；
3. 对引脚TIM2_<t>_PWMx(t=1 ~ 4, x=A, B)的上升沿或下降沿计数（本例程中的计数对象为PE5
   上升沿）；

说明：
1. 当选择的计数对象为TIMER6的上溢事件或下溢事件时，不需要设置Timer2的事件寄存器；计数对
   象为除TIMER6之外的外设产生的事件时，需要设置Timer2的事件寄存器。
2. 计数次数为10次，当达到计数次数时，Timer2的计数匹配标志位置位，如果使能了中断，则产生
   计数匹配中断；当前计数次数也可通过读取计数值寄存器获取。
