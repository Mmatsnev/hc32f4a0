平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要是系统时钟切换，并128分频后，通过MCO1(PA8)输出。
通过按键选择系统时钟：
SW1: MRC   (8MHz)      62.5 KHz
SW2: HRC   (16MHz)     125  KHz
SW3: LRC   (32KHz)     250  Hz
SW4: XTAL  (8MHz)      62.5 KHz
SW5: XTAL32(32.768KHz) 256  Hz
SW6: PLLH  (240MHz)    1.875MHz

================================================================================
使用步骤
================================================================================
1）连接PA8与示波器；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能，全速运行；
4) 通过按键SW1~SW6切换系统时钟频率；
5）查看示波器,可观察到波形及频率变化。
