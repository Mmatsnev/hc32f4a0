平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示了FMAC对ADC采样结果进行滤波，并在滤波完成中断里输出滤波结果，滤波结果
的精度可以通过结果右移来调节。用户可通过对PA02端口输入需要滤波的信号，并设置滤波
系数和滤波阶数，进行FIR滤波。
说明：
滤波阶数设置为n阶，输出结果会在n+1次之后趋于稳定。样例中阶数设置为10阶，输出结果
在第11次之后会趋于稳定。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
串口调试助手(115200bsp,8N1)

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）对PA02口输入滤波信号，设置滤波系数和滤波阶数以及结果移位数；
3）启动IDE的下载和调试功能，全速运行；
4）串口助手查看打印结果；


================================================================================
注意
================================================================================
若系数太小，需先对系数进行归一化再输入。
