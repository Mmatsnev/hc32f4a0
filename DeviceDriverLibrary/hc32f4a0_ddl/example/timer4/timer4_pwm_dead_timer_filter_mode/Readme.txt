平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示如何使用Timer4PWM死区定时器滤波模式。

说明：
样例配置比较输出模块OCO低通道，并通过PWM死区定时器滤波模式，过滤掉脉宽低于滤波计
数值宽度的OCO波形，输出其PWM波形。

================================================================================
使用步骤
================================================================================
1）测试板J1-E9(TIM4_1_OUH)、J1-E8(TIM4_1_OUL)引脚与示波器相连；
2）编译、下载并运行；
3）观察示波器，J1-24、J1-23通道波形频率为16Hz，占空比为1/2，极性相反。
