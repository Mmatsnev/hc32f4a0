平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例展示INTC模块EXINT检测功能。

说明：
本样例通过开发板上的KEY10引脚，展示了全局中断、分组中断、共享中断三种实现方式。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1) 选择打开line75~77之一，并重新编译；
2）启动IDE的下载程序并运行；
3）按下KEY10
   如#define GLOBAL_IRQ   (1U) -->LED_BLUE交替亮灭；
   如#define GROUP_IRQ    (1U) -->LED_YELLOW交替亮灭；
   如#define SHARE_IRQ    (1U) -->LED_RED交替亮灭。

================================================================================
注意
================================================================================
1. 全局中断使用入口No.0~31中的No.0；
2. 分组中断使用入口No.32~39中的No.33;
3. 共享中断使用入口No.128。
