功能描述
================================================================================
本例程实现了利用两个测试板,让SPI外设分别工作在主机/从机模式下通过DMA方式进行数
据收发功能。
通过宏定义可以配置为三线模式和四线模式，分别工作在SPI mode0（仅四线模式），
mode1，mode2（仅四线模式），mode3。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1) 准备两个板子进行硬件连接,使用EVB板子上的J9
      Master（测试板1）  Slave(测试板2)
NSS   TX                 TX
SCK   CK                 CK
MOSI  CTS                CTS
MISO  RTS                RTS
      GND                GND
2) 打开工程修改宏定义为#define APP_TEST_MODE  COM_MASTER，编译代码，下载进测试板1，全速运行。
3) 修改宏定义为#define APP_TEST_MODE  COM_SLAVE，编译代码，下载进测试板2，全速运行。
4) 短按测试板1的按键SW10，启动主机和从机之间全双工的数据收发。
5) 数据收发完成后，如果通信成功BLUE LED（LED2）闪烁，如通信失败RED LED（LED0）闪烁。


================================================================================
注意
================================================================================
注意步骤3)和4)的顺序。
