功能描述
================================================================================
本例程实现了SPI两个外设单元分别工作在主机/从机模式下,主机和从机之间通过查询方式
进行数据收发功能。
通过宏定义可以配置为三线模式和四线模式，分别工作在SPI mode0（仅四线模式），
mode1，mode2（仅四线模式），mode3。

无

================================================================================
使用步骤
================================================================================
1）在STK板子内硬件连接
      Master（SPI1）   Slave（SPI2）
NSS   PA4              PB12
SCK   PA5              PB10
MOSI  PC1              PC3
MISO  PF6              PB2
2) 打开工程并重新编译,下载代码全速运行。
3）全速运行后，短按按键SW10，代码进行主机和从机之间进行全双工的数据收发。
4) 数据收发完成后，如果通信成功蓝色LED闪烁，如通信失败红色LED闪烁。
