功能描述
================================================================================
本样例展示SPI模块读写Flash数据功能。

说明：
本样例设置SPI为3线式mode 3,每按一次SW2则触发一次擦除flash、写入数据到flash、从
flash读取数据，再讲读写的数据进行比较，相同则蓝色LED亮，不同则红色LED亮，每次按
键都将flash地址偏移到下一个扇区。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）短按SW10触发flash擦除、写入、读出后比较功能；
4）比较结果相同则蓝色LED亮，不同则红色LED亮；
5）重复短按SW10改变flash读写地址，再执行3、4的操作。
