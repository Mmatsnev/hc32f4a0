平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程实现了用QSPI对W25Q64进行读写。系统时钟240MHz，QSPI时钟为HCLK的3分频——80MHz。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------
USB转串口工具

辅助软件:
---------------------
串口调试助手软件（115200bps，8N1）

================================================================================
使用步骤
================================================================================
1）如果需要用在上位机观察程序运行情况，请将USB转串口工具正确地与开发板上的J28端口
  （靠近Micro USB接口）连接；如果不需要，请忽略此步骤；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）用设置断点的方式查看程序运行情况，或用USB转串口结合串口调试助手软件，在上位机
   观察；
