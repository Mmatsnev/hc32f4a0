平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本样例主要展示AES加密，解密功能，在程序中写入一组需要加密的数据和密钥，串口打印
加密或者解密后的数据。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------
无

辅助软件:
---------------------
串口调试助手(115200bsp,8N1)

================================================================================
使用步骤
================================================================================
1） 打开工程编译并全速运行。
2） 串口调试助手查看加密和解密后的数据。


================================================================================
注意
================================================================================
密钥的长度只能为128位，192位，或者256位。
