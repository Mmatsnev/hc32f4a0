平台说明
================================================================================
GCC工程，由Eclipse IDE外挂GNU-ARM Toolchain，再结合pyOCD GDB Server实现工程的编译、
链接和调试。在用Eclipse导入工程后，请将xxxx_PyOCDDebug中pyocd-gdbserver和SVD文件
设置为正确的路径；请将xxxx_PyOCDDownload中pyocd设置为正确的路径。注意，这些路径不
能包含非英文字符。

================================================================================
功能描述
================================================================================
本例程以PGA1、PGA3和PGA4为例，介绍了PGA的配置方法。例程设置PGA1（PA0）放大倍数为2.286，
PGA2（PA2）为6.4，PGA4（PA6）为10.667，并用串口将模拟输入信号放大前和放大后的电压值输出，
方便对比。

测试环境
================================================================================
测试用板:
---------------------
EV_F4A0_LQ176_V10

辅助工具:
---------------------

辅助软件:
---------------------
串口调试助手软件（115200bps，8N1）

================================================================================
使用步骤
================================================================================
1）如果需要用在上位机观察程序运行情况，请将USB转串口工具正确地与开发板上的J28端口（靠近
   MicroUSB接口）连接；如果不需要，请忽略此步骤；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）用杜邦线将靠近电位器的J10端口的AD3与各PGA对应的引脚短接，以便用电位器调节输入电压；
5）全速运行程序或在需要的位置设置断点；
6）可断点观察或用串口调试助手观察模拟输入信号经PGA放大前后的电压值。
